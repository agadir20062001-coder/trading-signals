## 📌 ما يفعله النظام

يراقب الأسواق المالية باستمرار، يحلل كل رمز بنقاش حقيقي بين 5 وكلاء ذكاء اصطناعي، ويرسل إشارات تداول مكتملة على تيليغرام — تلقائياً كل 15 دقيقة، أو فوراً عند طلبك.

## ⚙️ كيف يعمل النظام

### مرحلة البيانات
```
1H  ← RSI · MACD · Bollinger · MA20/50/200 · ATR · حجم · تقلب
4H  ← ADX · Stochastic · 12 نمط شمعة يابانية
1D  ← دعم/مقاومة · 10 أنماط مخططية (H&S · Cup · Wedge · Flag...)
15M ← تأكيد توقيت الدخول (معزز للثقة فقط)
```

### مرحلة النقاش (القلب)
```
الجولة 1 — كل وكيل يقدم تحليله وموقفه (BUY/SELL/HOLD) + 3 أدلة
الجولة 2 — كل وكيل يرد على المعارضين، قد يغيّر رأيه
الحكم    — القاضي يزن الحجج بالقوة والكثرة، يصدر قراراً مبرراً
```

الوكلاء الخمسة:
| الوكيل | تخصصه |
|--------|--------|
| المحلل التقني 📊 | مؤشرات + أنماط |
| محلل الأخبار 📰 | أحداث + محفزات |
| محلل المشاعر 💬 | زخم + خوف/جشع |
| محلل الأساسيات 📈 | PE · EPS · نمو |
| محلل الاقتصاد الكلي 🌍 | فائدة · تضخم · دورة |

### مرحلة القرار
```
إجماع استثنائي  5/5 ثقة ≥80%  → 🔥 إشارة فورية
إجماع قوي       4/5 ثقة ≥65%  → ✅ إشارة
إجماع مقبول     3/5 ثقة ≥55%  → ⚠️ إشارة محدودة
أقل من ذلك                    → ❌ صمت
```

## 🚀 خطوات التشغيل

### الخطوة 1 — الحصول على المفاتيح

**A. Groq API (مجاني — بدون بطاقة):**
```
1. https://console.groq.com
2. سجّل بإيميل → API Keys → Create Key
3. انسخ المفتاح (يبدأ بـ gsk_)
```

**B. بوت تيليغرام:**
```
1. ابحث عن @BotFather في تيليغرام
2. /newbot → اختر اسماً واسم مستخدم
3. انسخ التوكن (مثال: 7123456789:AAF_xxx)
4. ابحث عن @userinfobot → أرسل له أي رسالة → انسخ id
```

**C. FRED API (مجاني — بدون بطاقة):**
```
1. https://fred.stlouisfed.org/docs/api/api_key.html
2. Request API Key → سجّل → انسخ المفتاح
```

## 🔧 التخصيص

**تغيير عتبة الإجماع:**
```

```

LLM_MODEL=meta-llama/llama-3.3-70b-instruct:free
